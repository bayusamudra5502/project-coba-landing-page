Kenapa belajar di Cloudev?

Kami menyediakan beberapa fitur yang dapat membantu anda dalam mempelajari web development, diantaranya adalah sebagai berikut.

Materi terbaru dan telah direview dengan para ahli

Materi yang berada di kelas ini telah direview dan ditelaah oleh para ahli yang kompeten di bidangnya. Materi pun akan disesuaikan dengan perkembangan teknologi terkini.

Project yang akan direview oleh ahli

Pada tiap kelas akan diadakan sebuah mini project yang akan direview oleh para ahli dibidangnya. Hal ini akan membantu anda lebih berkembang untuk membuat website yang lebih profesional.

Punya pertanyaaan? Kami akan bantu jawab

Staff ahli kami akan berusaha untuk selalu membantu dan menjawab pertanyaan-pertanyaan anda dalam proses belajar di Cloudev.

Live Testing pada tiap kode

Anda akan dapat mengubah contoh kode yang kami berikan pada materi agar dapat melihat lebih dalam dan bereksprolasi dengannya.

Latihan dan Quiz

Kami akan meyediakan latihan-latihan yang berkaitan dengan materi yang anda pelajari baik secara teoritis dan praktik. Selain itu, tiap materi dan kelas akan ada sebuah kuis dan ujian untuk mengukur kemampuan anda.

Kelas di Cloudev

Kami menyediakan beberapa kelas yang dapat anda ambil untuk mempelajari web development dan desain.

Kelas Web Development 101

Kelas ini ditujukan untuk anda yang ingin mempelajari kelas pembelajaran web development dasar secara cepat. Pada kelas ini, anda akan mempelajari dasar HTML, CSS, dan Javascript dasar hingga anda bisa membuat halaman web sederhana.

Daftar

Kelas HTML

Kelas ini ditujukan untuk anda yang ingin serius untuk mempelajari dasar dari HTML. Pada kelas ini, anda akan akan mempelajari tag-tag HTML secara lengkap. Diharapkan anda telah mengikuti seri kelas Web Development 101.

Daftar

Kelas CSS

Kelas ini ditujukan untuk anda yang ingin serius mendalami CSS. Pada kelas ini akan diberikan tips-tips yang bermanfaat dan teknologi yang dapat membantu anda dalam membuat tampilan web yang menarik. Diharapkan anda telah mengikuti kelas Web Development 101.

Daftar

Kelas Javascript

Kelas ini ditujukan bagi anda yang ingin mendalami bahasa pemrograman Javascript untuk frontend. Pada kelas ini akan dibahas dimulai dari dasar-dasar javascript, javascript modern, ES6, DOM, dan lainnya. Diharapkan anda telah mengikuti kelas Web Development 101.

Daftar

Kelas Web Design 101

Kelas ini merupakan kelas dasar untuk memulai karir anda dalam web design. Pada kelas ini akan dibahas mengenai dasar dan tahapan-tahapan dalam desain halaman web.

Daftar

Kelas UI-UX

Kelas ini ditujukan bagi anda yang ingin lebih mendalami teknik-teknik dalam UI/UX. Pada kelas ini akan dipelajari tips-tips ampuh membuat halaman web yang baik dan bagus. Diharapkan anda telah memiliki dasar dari kelas Web Design 101 sebelumnya

Pertanyaan Umum

Berikut ini adalah pertanyaan-pertanyaan umum yang sering ditanyakan oleh orang-orang mengenai program ini

Berapa biaya untuk mengikuti program ini?
Program pembelajaran di Cloudev bersifat gratis. Anda tidak akan dipungut biaya apapun untuk mengikuti program ini. Hanya saja akan ada biaya komitmen sebesar Rp100.000 agar anda menyelesaikan program ini dengan baik.
Apakah orang awam boleh mengikuti program ini?
Tentu saja. Anda bisa mengikuti kelas 101 sebagai pijakan awal untuk menyelam di dunia web development. Akan tetapi, jika anda ingin mengikuti kelas yang lainnya, setidaknya anda telah memenuhhi prasyarat kelas tersebut agar proses pembelajaran berlangsung efektif.
Berapa lamakah program ini berjalan?
Program ini akan dijalankan selama 1 bulan. Anda akan belajar secara intensif oleh mentor-mentor berpengalaman agar anda memiliki pemahaman yang fresh mengenai web programming.
Apakah di akhir pembelajaran saya akan mendapatkan pekerjaan?
Kami akan berusaha membantu anda mencari pekerjaan anda sebagai web developer. Kami akan mempertemukan dengan rekan kerja kami sehingga anda dapat mendapatkan pekerjaan yang anda inginkan.
